# 🔐 Password Generator

A responsive and customizable password generator built with HTML, CSS, and JavaScript.

This project allows users to generate secure random passwords based on selected criteria such as password length, uppercase and lowercase letters, numbers, and special characters.

## 🌐 Live Demo

👉 https://password-generator-ten-pi-11.vercel.app/

## 🚀 Features

- Generate secure random passwords
- Customize password length
- Include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Copy password to clipboard with one click
- Responsive and clean interface

---

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript

## 📂 Getting Started

Clone the repository:

```bash
git clone https://github.com/Hyouem/password-generator.git
```

Open the project folder and run the `index.html` file in your browser.
